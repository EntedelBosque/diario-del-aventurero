# DEV-SPEC-018 — THE DIRECTOR ENGINE
> El **Director** es un Motor (NO el Oráculo): no genera narrativa, no interpreta el diario, no responde al jugador.

## Responsabilidad
Analiza el estado global del mundo y **propone** cambios. **Nunca ejecuta** cambios directamente.

## Áreas de análisis
Progreso, contratos, misiones, Grandes Destinos, bosses, relaciones, entidades, gremios, logros, historial.

## Propuestas (reglas de cada una en docs aparte)
Generar misiones/contratos, actualizar prioridades, proponer evolución de una entidad, proponer nuevo Título Legendario.

## Alcance
Define responsabilidades/límites/comunicación. NO algoritmos, criterios, fórmulas ni balance.
