# CODEX-010 - Implementacion de gremios y especializaciones

Todo Aventurero recibe los seis gremios oficiales. Una actividad puede repartir experiencia entre varios gremios con pesos enteros que suman 100. La experiencia y maestria de gremio nunca disminuyen, y cada premio se registra en historial.

CODEX-010 no define la formula de niveles, rangos ni titulos de gremio. El Motor registra experiencia y maestria, pero no calcula nuevos niveles o titulos hasta que el Developer Specification los defina.
